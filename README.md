# Challenge: Conversor de Moedas em Java

Este projeto é um **Conversor de Moedas desenvolvido em Java**, criado como parte do **Programa ONE – Oracle Next Education**.

A aplicação funciona no **console** e permite ao usuário converter valores entre diferentes moedas, utilizando **taxas de câmbio reais** obtidas a partir de uma **API externa**.

---

## Funcionalidades

- Conversão entre as moedas:
    - USD ⇄ BRL
    - EUR ⇄ BRL
    - USD ⇄ EUR
- Consumo de API externa de taxas de câmbio
- Menu interativo no console
- Histórico das conversões realizadas
- Valores exibidos com **duas casas decimais**

---

## Tecnologias utilizadas

- Java 17
- HttpClient (Java)
- Biblioteca Gson
- API ExchangeRate
