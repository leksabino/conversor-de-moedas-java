# Conversor de Moedas em Java

Este projeto é um conversor de moedas desenvolvido em Java como parte do Programa ONE (Oracle Next Education).

A aplicação funciona via console e permite ao usuário converter valores entre diferentes moedas utilizando dados reais obtidos de uma API externa de câmbio.

## Funcionalidades

- Conversão entre as moedas:
    - USD ⇄ BRL
    - EUR ⇄ BRL
    - USD ⇄ EUR
- Consumo de API externa de taxas de câmbio
- Menu interativo no console
- Histórico das conversões realizadas
- Exibição dos valores convertidos com duas casas decimais

## Tecnologias utilizadas

- Java 17
- HttpClient (Java)
- Biblioteca Gson
- API ExchangeRate
